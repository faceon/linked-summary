**Linked Summary** is a Chrome extension powered by built-in AI.

## Installation

- `npm install`
- `npm run build`
- In Chrome, open `chrome://extensions/`
- Toggle on **Developer mode** (top-right)
- Click **Load unpacked** and select the `dist/` directory
